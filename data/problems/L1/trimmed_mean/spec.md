Write a function trimmed_mean(nums) that returns the arithmetic mean of a list of numbers after removing exactly one smallest value and one largest value. If the list has fewer than 3 items, return 0. If there are duplicate minimum or maximum values, remove only one occurrence of each. The result should be a float. The input list must not be modified. Examples: [1,2,3] -> 2.0, [1,1,9,10] removes one 1 and one 10, leaving [1,9].
