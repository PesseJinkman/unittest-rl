Write a function rotate_right(nums, k) that returns a new list with the elements of nums rotated to the right by k positions. Do not modify the input list. If nums is empty, return an empty list. k may be any integer: treat negative k as rotating left, and reduce large values using the list length. Examples: rotate_right([1,2,3,4], 1) -> [4,1,2,3], rotate_right([1,2,3,4], -1) -> [2,3,4,1].
