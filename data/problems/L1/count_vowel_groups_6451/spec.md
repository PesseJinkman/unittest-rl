Write a function count_vowel_groups(text) that returns the number of contiguous groups of vowels in a string. Treat vowels as the letters a, e, i, o, u in either lowercase or uppercase. Consecutive vowels count as one group, so "queue" has 1 group. Any non-vowel breaks a group. Return 0 for an empty string or a string with no vowels. Only alphabetic vowel characters count; all other characters are simply separators.
