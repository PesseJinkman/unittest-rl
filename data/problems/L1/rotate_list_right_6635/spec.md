Write a function rotate_right(items, k) that returns a new list with the elements of items rotated to the right by k positions. Do not modify the input list. If items is empty, return an empty list. k may be any integer: values larger than the list length should wrap around, and negative k should rotate left accordingly. Examples: rotate_right([1,2,3,4], 1) -> [4,1,2,3], rotate_right([1,2,3,4], -1) -> [2,3,4,1].
