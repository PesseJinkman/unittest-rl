Write a function rotate_left(items, k) that returns a new list with the elements of items rotated left by k positions. Rotation wraps around from the front to the end. Do not modify the input list. If items is empty, return an empty list. k may be zero, larger than the list length, or negative; treat negative k as rotating right, using normal wraparound. Examples: rotate_left([1,2,3,4],1) -> [2,3,4,1], rotate_left([1,2,3],-1) -> [3,1,2].
