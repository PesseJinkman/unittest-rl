Write a function count_vowel_groups(text) that returns the number of contiguous vowel groups in a string. Treat vowels as only a, e, i, o, u in either lowercase or uppercase. Non-vowel characters break groups. Return 0 for the empty string or strings with no vowels. Do not modify the input.
