Write a function longest_common_prefix(words) that returns the longest starting substring shared by every string in the list. If words is empty, return an empty string. If any string is empty, the result is empty unless all compared characters before that point match and the empty string ends first, so the result is still empty. Comparison is case-sensitive. The function should work for lists of one or more strings and return the single string itself when the list has one element.
