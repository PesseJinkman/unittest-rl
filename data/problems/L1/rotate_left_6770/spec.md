Write a function rotate_left(items, k) that returns a new list with the elements of items rotated to the left by k positions. Do not modify the input list. If items is empty, return an empty list. k may be any integer: values larger than the list length should wrap around, and negative k should rotate in the opposite direction using normal modulo behavior. Examples: rotate_left([1,2,3,4], 1) -> [2,3,4,1], rotate_left([1,2,3], -1) -> [3,1,2].
