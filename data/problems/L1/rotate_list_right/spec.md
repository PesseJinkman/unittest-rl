Write a function rotate_right(items, k) that returns a new list with the elements of items rotated to the right by k positions. Do not modify the input list. If items is empty, return an empty list. k is a non-negative integer and may be larger than the list length; in that case, rotate by k modulo len(items). Rotating by 0 or by a multiple of the list length returns a copy of the original order.
