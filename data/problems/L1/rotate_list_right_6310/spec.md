Write a function rotate_list_right(items, k) that returns a new list with the elements of items rotated to the right by k positions. Do not modify the input list. If items is empty, return an empty list. k is a non-negative integer and may be larger than the list length; in that case rotate by k modulo len(items). A rotation by 0 leaves the list unchanged. The function should work for lists of any primitive values, including duplicates.
