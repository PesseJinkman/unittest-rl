Implement a RollingCounter class for integer timestamps. Construct with a positive window size. add(ts, value=1) records a count at timestamp ts; ts must be >= the most recent added ts, else raise ValueError. Multiple adds at the same ts accumulate. total() returns the sum of counts whose timestamps are in the inclusive range [latest_ts-window+1, latest_ts]. count_at(ts) returns the stored count exactly at ts if still retained, else 0. Old data must be discarded as new timestamps advance. If no events were added, total() and count_at(...) return 0.
