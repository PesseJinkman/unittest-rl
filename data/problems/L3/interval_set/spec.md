Implement an IntervalSet class for inclusive integer intervals. State is a sorted, non-overlapping list of ranges. add(start,end) inserts an interval, swapping bounds if start>end, and merges any overlaps or touching intervals (e.g. [1,2] and [3,4] become [1,4]). remove(start,end) deletes all covered values, possibly splitting intervals; swapped bounds also apply. contains(x) returns True if x lies in any stored interval. ranges() returns a deep copy of the current intervals as [start,end] pairs in sorted order. Operations are order-dependent and must preserve invariants after every call.
