Implement class RollingCounter(window_size). It stores integer timestamped deltas in insertion order. window_size must be > 0 or raise ValueError. add(ts, delta=1) appends an event; timestamps must be non-decreasing relative to prior adds or raise ValueError. total_at(ts) returns sum of deltas for events with timestamp in [ts-window_size+1, ts]. Query ts may be any int, but must not be less than the last added timestamp or raise ValueError. Expired events should be discarded during queries/adds so state shrinks over time. reset() clears all state and returns None.
