Implement a RollingWindowStats class for a fixed-size window of the most recent integers. Constructor takes capacity>0 or raises ValueError. add(x) appends x and evicts the oldest item if size exceeds capacity, returning the current number of stored items. mean() returns the arithmetic mean as float, or raises ValueError if empty. minmax() returns (min,max), or raises ValueError if empty. values() returns a new list of current items from oldest to newest. clear() empties the window and returns 0. Behavior is order-dependent and must stay correct across many evictions.
