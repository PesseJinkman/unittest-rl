Implement an IntervalSet class storing inclusive integer intervals, always kept sorted and non-overlapping. add(start,end) inserts an interval; if start>end, swap them. New intervals merge with any existing interval that overlaps or is directly adjacent (e.g. [1,2] with [3,4] -> [1,4]). remove(start,end) deletes covered values; if start>end, swap them. Removal may shrink or split intervals. contains(x) returns True if x lies in any interval. spans() returns a deep copy of the current interval list as [(start,end),...]. Mutations are order-dependent; invariants must hold after every operation.
