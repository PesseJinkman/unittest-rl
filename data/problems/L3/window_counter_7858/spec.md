Implement a WindowCounter class that tracks integer events in a fixed-capacity rolling window of the most recent values. WindowCounter(capacity) requires capacity>0. add(x) appends x and evicts the oldest item if size would exceed capacity, returning the current window sum. total() returns the current sum. average() returns the arithmetic mean as float, or raises ValueError if empty. last(n) returns a list of the most recent n items in order from oldest to newest among those returned; if n<0 raise ValueError; if n exceeds size, return the whole window.
