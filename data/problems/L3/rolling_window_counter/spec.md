Implement class RollingCounter(window_size) for counting recent integer events. window_size must be > 0 or raise ValueError. State is order-sensitive. add(timestamp, value=1) appends an event; timestamps must be non-decreasing across adds or raise ValueError. value may be any int, including negative or zero. total(now=None) returns the sum of values whose timestamps are within the inclusive window [T-window_size+1, T], where T is the latest added timestamp if now is None, otherwise the provided now. query(now) is like total(now) but raises ValueError if now is less than the latest pruned/query time already used.
