Implement class RollingMedianWindow(size) for a fixed positive window size. add(value) appends an int and returns the current median of the stored values. If the window exceeds size, evict the oldest value first. median() returns the current median without changing state, or raises ValueError if empty. For odd count, median is the middle value; for even count, return the arithmetic mean as float. values() returns the current window contents from oldest to newest. Duplicate and negative numbers are allowed.
