Implement a RollingStats class for a fixed-size sliding window of numbers. Constructor takes capacity>0. add(x) appends x; if full, it evicts the oldest value. mean() returns the arithmetic mean of current items, raising IndexError if empty. min() and max() return current minimum/maximum, also raising IndexError if empty. values() returns the current window contents from oldest to newest as a new list. clear() empties the window. All methods are order-dependent; duplicate values and negative numbers must be handled correctly.
