Implement an IntervalSet class storing inclusive integer intervals, always normalized as sorted, non-overlapping, and non-adjacent. add(a,b) inserts [min(a,b), max(a,b)] and merges overlaps or touching intervals. remove(a,b) deletes all points in that range, possibly splitting intervals. contains(x) returns True if x is covered. total_covered() returns the count of covered integers. ranges() returns a new list of [start,end] pairs. Empty state is allowed. Reversed endpoints must be accepted. Removing a disjoint range changes nothing.
