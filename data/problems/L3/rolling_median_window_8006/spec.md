Implement a RollingMedian class for a fixed-size sliding window of numbers. Constructor takes positive integer size; size<=0 raises ValueError. Method add(value) appends a number and returns current median of the window after insertion. While fewer than size items have been seen, median is over all seen items. Once full, each add evicts the oldest item first-in-first-out. For odd count, median is the middle sorted value; for even count, median is the arithmetic mean as float. Method snapshot() returns the current window contents from oldest to newest as a new list.
