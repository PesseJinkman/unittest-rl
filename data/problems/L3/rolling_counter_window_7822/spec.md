Implement a RollingCounter class for per-key counts inside a sliding time window. Construct with a positive integer window size. add(ts,key,amount=1) records an event at integer timestamp ts with positive amount. Timestamps across all add/query calls must be non-decreasing; otherwise raise ValueError. total(ts,key=None) returns the sum of amounts with timestamp > ts-window and <= ts, either for one key or all keys if key is None. Old events must expire as time advances. Unknown keys return 0. Zero/negative window or amount raise ValueError.
