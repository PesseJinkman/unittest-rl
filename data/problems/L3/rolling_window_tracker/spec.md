Implement a RollingWindow class tracking the last N added integers. Constructor takes positive capacity; otherwise raise ValueError. add(x) appends x and evicts the oldest item if size exceeds capacity. total() returns sum of current items, average() returns float mean and raises ZeroDivisionError when empty, values() returns a new list in oldest-to-newest order. resize(new_capacity) changes capacity, raising ValueError if nonpositive; if shrinking, evict oldest items until size fits. State changes are order-dependent and sums must stay consistent after many adds/resizes.
