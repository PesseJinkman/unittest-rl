Implement a WindowCounter class tracking integer events in the last N ticks. Constructor WindowCounter(window) requires window>0. Method add(value=1) records one event at the current tick, increasing that tick’s count by value; negative values raise ValueError. Method advance(steps=1) moves time forward by positive integer steps, expiring old ticks; returns current total in window. Method total() returns the sum for ticks in [current-window+1,current]. Calling add multiple times before advancing accumulates on the same tick. Advancing by many steps may clear all history.
