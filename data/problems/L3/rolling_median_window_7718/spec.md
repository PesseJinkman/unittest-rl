Implement class RollingMedian(window_size) for a positive integer size. It maintains the last at most window_size added numbers in insertion order. add(x) appends x and returns the current median of stored values. median() returns the current median without changing state, or raises IndexError if empty. For an odd count, median is the middle value after sorting; for an even count, it is the arithmetic mean of the two middle values as float. Oldest items are discarded once capacity is exceeded. clear() removes all items. Duplicates and negative values are allowed.
