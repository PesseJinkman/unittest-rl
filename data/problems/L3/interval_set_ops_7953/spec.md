Implement an IntervalSet class storing a normalized set of inclusive integer intervals. add(start,end) inserts an interval, merging overlaps and touching ranges (e.g. [1,2] with [3,4] -> [1,4]). remove(start,end) deletes covered values and may split intervals. contains(x) returns whether x is present. spans() returns the current normalized intervals as a list of (start,end) tuples. If start>end, methods must raise ValueError. State is order-dependent but representation invariant is sorted, non-overlapping, non-touching intervals.
