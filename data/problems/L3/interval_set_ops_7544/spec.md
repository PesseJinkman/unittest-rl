Implement an IntervalSet class storing closed integer intervals, kept normalized as sorted, non-overlapping, and non-adjacent. add(start,end) inserts an interval, merging overlaps and touching neighbors; if start>end swap them. remove(start,end) deletes covered points, possibly splitting intervals; also swap if reversed. contains(x) returns whether x lies in any stored interval. intervals() returns a new list of [start,end] pairs. Invariants must hold after every mutation, and operation order matters.
