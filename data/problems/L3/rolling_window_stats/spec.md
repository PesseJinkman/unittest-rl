Implement a RollingWindow class for integers with fixed positive capacity. Methods: add(x) appends x and evicts the oldest item if over capacity, returning the current average as float, or None if empty. stats() returns a dict {"count", "sum", "min", "max", "avg"}; for an empty window use None for min/max/avg. snapshot() returns the current items from oldest to newest. Capacity <= 0 must raise ValueError at construction. State is order-dependent: evictions happen exactly when size exceeds capacity, and min/max must stay correct after evicting the previous extreme.
