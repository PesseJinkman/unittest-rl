Implement a RollingWindow class over the last N pushed integers. Constructor takes positive capacity or raises ValueError. push(x) appends an int and evicts the oldest when size exceeds capacity, returning the current count. stats() returns None when empty, else a dict with keys count, sum, min, max, avg where avg is sum/count as float. pop_oldest() removes and returns the oldest value, raising IndexError if empty. State is order-dependent; stats must always reflect only the current window after any sequence of pushes and pops.
