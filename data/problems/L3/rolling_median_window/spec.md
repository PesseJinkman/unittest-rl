Implement a RollingMedian class over a fixed positive window size. add(x) appends a number and returns the current median of the last at most size values. median() returns the current median without changing state, or raises ValueError if empty. For an odd count, the median is the middle value after sorting; for an even count, it is the arithmetic mean of the two middle values as a float. Only the most recent size items are kept, so results depend on call order. __len__ returns the current number of stored items.
