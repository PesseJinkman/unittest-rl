Implement a WindowCounter class for rolling sums by integer timestamps. Construct with positive window size. add(ts, value) requires ts >= last added ts, else raises ValueError; it stores the event and returns current sum of values with timestamps in [ts-window+1, ts]. total(ts=None) returns the sum in the active window at the latest timestamp, or as of a provided ts using the same inclusive range; querying with ts before the latest added timestamp raises ValueError. Expire old events on add/total. Empty state totals to 0. Negative values are allowed.
