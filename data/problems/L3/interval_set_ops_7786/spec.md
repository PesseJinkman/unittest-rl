Implement an IntervalSet class storing disjoint half-open integer intervals [start,end). add(start,end) inserts an interval, merging any overlapping or touching intervals; if start>=end, do nothing. remove(start,end) deletes that range from existing intervals, possibly splitting them; if start>=end, do nothing. contains(x) returns True iff x is in any stored interval. spans() returns the current intervals as a list of (start,end) tuples in ascending order. Invariants: intervals remain sorted, disjoint, and never empty; touching intervals must be merged.
