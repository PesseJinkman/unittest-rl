Implement a RollingCounter class for fixed-size time windows. `hit(ts, value=1)` records a non-negative integer value at integer timestamp `ts` and returns the current window sum after adding it. Timestamps for `hit` are non-decreasing; calling `hit` with smaller `ts` raises ValueError. `total(ts=None)` returns the sum of all values with timestamp in `[current-window+1, current]`, where `current` is the latest seen timestamp or the provided `ts`. Querying with `ts` smaller than latest seen raises ValueError. Window size is a positive integer. Expired items must be discarded as time advances.
