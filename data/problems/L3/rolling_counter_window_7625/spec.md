Implement a RollingCounter class over integer event times with a fixed positive window size. add(t, value=1) records a non-negative integer value at time t. Times must be non-decreasing across add calls, else raise ValueError. total(now=None) returns the sum of values with timestamps in the inclusive range [end-window+1, end], where end is the last added time if now is None, otherwise now. If now is provided and is earlier than the last added time, raise ValueError. Empty structure totals to 0. Values outside the window are evicted as time advances; repeated totals must not change results.
