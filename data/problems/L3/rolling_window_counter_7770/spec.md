Implement a HitWindow class tracking integer timestamps in nondecreasing order. It is created with a positive window size. add(ts, count=1) records count hits at timestamp ts and returns the total hits currently within the inclusive range [ts-window+1, ts]. count(ts=None) returns hits in the current window; if ts is None, use the latest seen timestamp, or 0 if empty. Timestamps passed to add must not go backwards; otherwise raise ValueError. count(ts) must not mutate state and may query past or future times relative to latest timestamp.
