Write a function rotate_left(items, k) that returns a new list with the elements of items rotated left by k positions. For example, rotating [1,2,3,4] by 1 gives [2,3,4,1]. If k is 0, return a shallow copy in the same order. If k is larger than the list length, wrap around using modulo. If k is negative, rotate right by abs(k), which is equivalent to left rotation by k modulo len(items). For an empty list, return an empty list.
