Write a function `trimmed_mean(nums)` that returns the arithmetic mean of a list of numbers after removing exactly one smallest value and exactly one largest value. If the list has fewer than 3 items, return 0.0. If there are duplicate minima or maxima, remove only one occurrence of each. Return the result as a float. The input list must not be modified. Examples: [1,2,3] -> 2.0, [5,5,5,5] -> 5.0, [1,10,2,9] -> 5.5.
