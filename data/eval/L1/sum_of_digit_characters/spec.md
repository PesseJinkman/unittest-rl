Write a function sum_of_digit_characters(text) that returns the sum of all decimal digit characters found in the input string. Each digit is added separately, so "a12b3" returns 6. Ignore all non-digit characters. If the string has no digits, return 0. The function should work for empty strings, repeated digits, whitespace, punctuation, and mixed letters/numbers. Only characters '0' through '9' count as digits.
