Write a function rotate_list_right(items, k) that returns a new list with the elements of items rotated to the right by k positions. Do not modify the input list. If items is empty, return an empty list. k may be any integer: treat negative k as rotating left, and values larger than the list length should wrap using modulo. Examples: rotate [1,2,3,4] by 1 -> [4,1,2,3], by -1 -> [2,3,4,1].
