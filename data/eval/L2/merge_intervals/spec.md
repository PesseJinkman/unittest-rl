Implement merge_intervals(intervals) to merge overlapping closed intervals. intervals must be a list of 2-item lists/tuples of integers [start, end]. Raise TypeError if intervals is not a list, any item is not a 2-item list/tuple, or bounds are not ints. Raise ValueError if any start > end. Return a new list of [start, end] pairs sorted by start, merging intervals when next.start <= current.end. Adjacent intervals like [1,2] and [3,4] do not merge. Empty input returns []. Do not mutate the input.
