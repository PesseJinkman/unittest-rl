Implement an IntervalSet class storing a normalized set of inclusive integer intervals. add(a,b) inserts all integers from min(a,b) to max(a,b), merging overlaps and touching intervals (e.g. [1,2] with [3,5] -> [1,5]). remove(a,b) deletes that inclusive range, possibly splitting intervals. contains(x) returns whether x is present. intervals() returns a new list of [start,end] pairs in sorted normalized form. Empty state starts with no intervals. Operations are order-dependent and must preserve the invariant: intervals are sorted, disjoint, and never touch.
