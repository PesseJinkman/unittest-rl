Implement an IntervalSet class storing disjoint half-open integer intervals [start,end). add(start,end) inserts a range, merging overlaps and touching ranges; empty or reversed ranges are ignored. remove(start,end) deletes coverage from that range, possibly splitting intervals; empty or reversed ranges do nothing. contains(x) returns True iff x is covered. intervals() returns the current intervals as a list of (start,end) tuples in sorted order. count() returns total covered length. Invariants: stored intervals stay sorted, disjoint, and non-touching after every operation.
