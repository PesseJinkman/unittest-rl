Implement an IntervalSet class storing non-overlapping half-open integer intervals [start,end). add(start,end) inserts a range, merging any overlaps or touching neighbors; invalid ranges with start>=end are ignored. remove(start,end) deletes that range, possibly splitting existing intervals; invalid ranges are ignored. contains(x) returns True iff x lies in any stored interval. intervals() returns the current intervals as a new sorted list of [start,end] pairs. State is order-dependent but the final representation must stay sorted and merged.
