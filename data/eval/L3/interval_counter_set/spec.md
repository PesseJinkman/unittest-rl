Implement an IntervalCounterSet class storing a normalized union of inclusive integer intervals. Methods: add(start,end) merges overlap or adjacency; remove(start,end) deletes covered points and may split intervals; contains(x) returns whether x is covered; total_covered() returns the number of covered integers. If start>end, swap them. Invariant: stored intervals are sorted, disjoint, and never adjacent. Operations are order-dependent. Removing an uncovered range changes nothing. Negative values are allowed.
