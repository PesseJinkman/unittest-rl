Implement a RollingWindowCounter class for integer timestamps. Construct with window_size > 0. add(ts, value=1) appends an event; timestamps must be non-decreasing or raise ValueError. Events older than current_ts-window_size are expired, so the active window is [current_ts-window_size, current_ts]. total() returns the sum of active values after the most recent add, or 0 before any add. count() returns number of active events. snapshot() returns (count,total). Zero and negative values are allowed. Expiration happens only on add; queries do not advance time.
